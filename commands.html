Music [12]

[Optional] {Required}

/play {Link / Song Name} | Play a track from youtube. Permission: [Everyone]
/back | Rewind to last track in queue. Permission: [Everyone]
/clearq | Clear all music from queue. Permission: [Everyone]
/join | Ask @Furnace to join VC. Permission: [Everyone]
/leave | Furnace will leave VC. Permission: [Everyone]
/lyrics | Display current song's lyrics. Permission: [Everyone]
/next | Skip to next song in queue. Permission: [Everyone]
/np | Display currently playing song. Permission: [Everyone]
/pause | Pause the music queue. Permission: [Everyone]
/queue | List all songs in the queue. Permission: [Everyone]
/stop | Stop the music player. Permission: [Everyone]
/volume | Set the volume of the player. Permission: [Everyone]

Moderation [11]

[Optional] {Required}

/ban { @User } [Reason] | Ban user from guild with optional reason. Permission: [BAN_MEMBERS]
/kick { @User } [Reason] | Kick user from guild with optional reason. Permission: [KICK_MEMBERS]
/modhistory { @User } | View modhistory of mentioned user. Permission: [MANAGE_MESSAGES]
/history { @User } | View punishment history of mentioned user. Permission: [MANAGE_MESSAGES]
/purge {quantity} | Bulk delete specified amount of messages from channel. Permission: [MANAGE_MESSAGES]
/rwarns { @User } | Reset all warnings of user. Permission: [MANAGE_MESSAGES]
/slowmode {time} | Set slowmode in channel. Permission: [MANAGE_CHANNELS]
/sudo { @User } | Sudo a user with a webhook. Permission: [MANAGE_WEBHOOKS]
/unban { @User } | Unban user from guild. Permission: [BAN_MEMBERS]
/unwarn { @User } {ID} | Remove warning ID from user. Permission: [MANAGE_MESSAGES]
/warn { @User } [Reason] | Warn user with optional reason. Permission: [MANAGE_MESSAGES]
